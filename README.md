# banking-app-csharp
Konsolenanwendung in C# zur Verwaltung einer Bank

## Das Ziel
Es soll eine Bankingapp in c# programmiert werden. Die App soll zum einen als Konsolenapp vollständig funktionsfähig laufen. Zum anderen soll später eine Webassembly dazukommen, die auf dieselben Funktionen zugreift.

## Vorgehen
1. Neues Projekt erstellen
2. Ersten Kontakt mit der App in der Konsole aufnehmen
